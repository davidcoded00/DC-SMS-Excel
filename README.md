▪ Features: Data export and search, working with .xlsx files 
Description: 
A simple application for exporting SMS messages to an Excel file. 
Features – How to Use the Application 
1. Export Messages – Tap the message icon to export all SMS messages to an Excel (.xlsx) 
file. 
2. Schedule – Tap the calendar icon to open the DateTimePickers, where you can select a 
start and end date. The app will export only the SMS messages sent or received within the 
selected date range. 
3. Export by Text – Enter a specific word or phrase in the text field, then tap the search icon. 
The app will export only the SMS messages that contain the entered text to the Excel file. 
4. Export Contacts – Tap the contacts icon to export all contacts saved in your phonebook 
to an Excel (.xlsx) file.
